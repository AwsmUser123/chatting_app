0x00 - Error code (with extra information).
0x01 - Initiate check sequence.
0x02 - OK.

0x03 - Registration data.
0x05 - Login data.

0x06 - Log out.
0x07 - Create a chat.
0x09 - Join a chat.
0x10 - List all chats.

0x0a - Quit current chat.
0x0b - Send a message.
0x0d - List all messages.

0x20 - Quit.
